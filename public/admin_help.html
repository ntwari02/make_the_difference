Setting Up Your Profile

Complete your admin profile to personalize your experience and ensure proper system access.

Step-by-Step Profile Setup:

  1. Access Account Settings: Click on your profile picture in the top-right corner and select "Account Settings"
  2. Personal Information: Fill in your full name, email, phone number, and position
  3. Profile Picture: Upload a professional photo (recommended size: 200x200px)
  4. Preferences: Set your timezone, language, and theme preferences
  5. Notifications: Configure email, SMS, and push notification settings
  6. Save Changes: Click "Save Changes" to update your profile
Pro Tip:

Keep your profile information up-to-date, especially your email address, as it's used for password resets and important notifications.

Creating and Managing Users

Learn how to create new users and manage existing user accounts effectively.

Creating New Users:

  1. Navigate to User Management: Go to the sidebar and click "User Management"
  2. Add New User: Click the "Add New User" button
  3. Fill User Details: Enter the required information:
    • Full Name (required)
    • Email Address (required, must be unique)
    • Password (minimum 8 characters)
    • Role (Admin, User, or Super Admin)
  4. Set Permissions: Assign appropriate permissions based on the user's role
  5. Save User: Click "Create User" to add the new user

Managing Existing Users:

  • View Users: Use the search and filter options to find specific users
  • Edit User: Click the edit icon to modify user information
  • Deactivate User: Use the status toggle to temporarily disable accounts
  • Delete User: Permanently remove users (use with caution)
  • Reset Password: Send password reset emails to users
Important:

Only Super Admins can create new admin users. Regular admins can only create regular users.

Creating Scholarships

Create and manage scholarship opportunities for students to apply.

Creating a New Scholarship:

  1. Access Scholarship Management: Go to "Scholarship Management" in the sidebar
  2. Create New Scholarship: Click "Add New Scholarship"
  3. Basic Information: Fill in the scholarship details:
    • Scholarship Name (required)
    • Description and eligibility criteria
    • Award amount and number of awards
    • Application deadline
    • Academic level and field of study
  4. Requirements: Specify required documents and minimum GPA
  5. Settings: Set scholarship status (active/inactive)
  6. Publish: Click "Create Scholarship" to make it available

Managing Scholarship Status:

  • Active: Scholarship is open for applications
  • Inactive: Scholarship is temporarily closed
  • Expired: Application deadline has passed
  • Archived: Scholarship is no longer available

Managing Applications

Review and process scholarship applications efficiently.

Application Review Process:

  1. Access Applications: Go to "Applications" in the sidebar
  2. Filter Applications: Use filters to view applications by:
    • Scholarship type
    • Application status (pending, approved, rejected)
    • Date range
    • Academic level
  3. Review Application: Click on an application to view details:
    • Personal information
    • Academic records
    • Uploaded documents
    • Motivation statement
  4. Make Decision: Approve, reject, or request additional information
  5. Add Notes: Include reviewer comments for internal reference
  6. Notify Applicant: Send automated email notifications

Application Statuses:

  • Pending: Application submitted, awaiting review
  • Under Review: Application is being evaluated
  • Approved: Application meets all criteria
  • Rejected: Application does not meet requirements
  • Waitlisted: Application may be considered if spots open

Login Issues

Common login problems and their solutions.

Common Login Problems:

"Invalid email or password"

Solutions:

  • Check that Caps Lock is off
  • Verify your email address is correct
  • Use the "Forgot Password" feature to reset your password
  • Contact support if the problem persists
"Account locked"

Solutions:

  • Wait 15 minutes for automatic unlock
  • Use "Forgot Password" to reset your password
  • Contact your system administrator
"Session expired"

Solutions:

  • Simply log in again with your credentials
  • Check "Remember Me" for longer sessions
  • Ensure your browser accepts cookies
Best Practices:
  • Use a strong, unique password
  • Enable two-factor authentication if available
  • Log out when using shared computers
  • Keep your browser updated

File Upload Problems

Resolve common file upload issues and learn best practices.

Common Upload Issues:

"File too large"

Solutions:

  • Compress the file using online tools
  • Convert to a smaller format (e.g., PDF instead of high-res images)
  • Split large documents into smaller parts
  • Maximum file size: 5MB for most uploads
"Invalid file type"

Solutions:

  • Use supported formats: PDF, DOC, DOCX, JPG, PNG
  • Check file extension is correct
  • Convert files to supported formats
  • Ensure file isn't corrupted
"Upload failed"

Solutions:

  • Check your internet connection
  • Try refreshing the page
  • Use a different browser
  • Contact support if problem persists

File Upload Best Practices:

  • File Naming: Use descriptive names without special characters
  • File Size: Keep files under 5MB for faster uploads
  • Format: Use PDF for documents, JPG/PNG for images
  • Quality: Ensure files are clear and readable
  • Backup: Keep copies of important files

Resetting User Passwords

Learn how to reset passwords for users who have forgotten their credentials.

Security Questions Setup:

  • During Registration: All new users must set up 3 security questions
  • Question Selection: Users choose from predefined security questions
  • Answer Storage: Answers are securely stored in the database
  • Setup Tracking: System tracks if security questions are configured

User Password Reset Process:

  1. Email Entry: User enters their email address on login page
  2. Security Questions: System displays user's security questions
  3. Answer Verification: User must answer at least 2 questions correctly
  4. Password Reset: User sets new password after verification
  5. Access Restored: User can log in with new password

Admin Password Reset:

  1. Access User Management: Go to "User Management" in the sidebar
  2. Find the User: Search for the user by name or email address
  3. Edit User: Click the edit icon for the specific user
  4. Change Password: Enter a new temporary password
  5. Notify User: Inform the user of their new temporary password
  6. Force Change: Set the password to expire on next login

Security Features:

  • Multiple Questions: Users must answer at least 2 of 3 questions
  • Case Insensitive: Answers are not case-sensitive for user convenience
  • Reset Tokens: Temporary tokens expire after 15 minutes
  • Answer Validation: System validates answers against stored responses
  • Audit Trail: Log all password reset attempts and security question usage
Security Note:

The system now uses security questions instead of email verification for password reset. Users without security questions set up must contact an administrator for assistance. Always use secure, temporary passwords and require users to change them on their next login.

Common Error Messages

Understand and resolve common error messages you may encounter while using the admin panel.

Authentication Errors:

"Session expired"

Your login session has timed out. Simply log in again to continue.

"Access denied"

You don't have permission to access this feature. Contact your administrator.

Data Entry Errors:

"Required field missing"

Fill in all required fields marked with an asterisk (*).

"Invalid email format"

Enter a valid email address (e.g., user@example.com).

"Password too weak"

Use a stronger password with letters, numbers, and symbols.

System Errors:

"Server error (500)"

A server error occurred. Try refreshing the page or contact support.

"Page not found (404)"

The requested page doesn't exist. Check the URL or use navigation menu.

Quick Fixes:
  • Clear your browser cache and cookies
  • Try using a different browser
  • Check your internet connection
  • Contact support if errors persist
